# Repository Agent Rules

## Git Commit Messages
- Follow the Emoji-Log convention from `https://github.com/ahmadawais/Emoji-Log`.
- Use exactly one leading category marker and then an imperative subject.
- Format: `<EMOJI> <TYPE>: <imperative summary>`
- Allowed prefixes:
  - `📦 NEW:`
  - `👌 IMPROVE:`
  - `🐛 FIX:`
  - `📖 DOC:`
  - `🚀 RELEASE:`
  - `🤖 TEST:`
  - `‼️ BREAKING:`
- Write the subject in imperative mood, for example `📦 NEW: Add sitemap support`.
- Choose the dominant action and do not combine multiple primary actions into one commit subject.
